秦导语录

Tan Jay / 2022-07-13

突然就理解孔子的弟子为何要编写《论语》了，老师思想太深刻，一时半会参不透呀，或者是有道理记不住呀，那就写下来吧。

灵魂一击

  1. 简单 $+$ 重复$=$成功.

老师译语: 简单重复$=$专注执着美好

  1. 优秀也可能因为是自卑.
  2. 追求完美有时也不好.
  3. 边做边学 $Learning \ by \ doing$ ，不会不要紧，更重要的是学习能力.
  4. 用人单位看重的是脑子，不全是你已经会什么.
  5. $Brilliant(t) = Ordinary(t-1) + Frustration(t-1);$
    $Life(t) = Brilliant(t) + Ordinary(t) + Frustration(t).$
